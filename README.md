# Create Appncy

Create projects as Goncy would.

## Installation

```bash
pnpm install -g appncy
```

## Usage

```bash
# Interactive mode
pnpm create appncy

# Override template
pnpm create appncy --template <template>

# Override path
pnpm create appncy --path <path>

# Get Command help
pnpm create appncy --help
```
